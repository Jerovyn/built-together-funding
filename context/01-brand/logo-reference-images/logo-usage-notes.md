Built Together Funding Logo Usage Notes

The final selected logo direction is based on the crossed wrench + hammer identity from the original old logo.

Use btf-logo-horizontal-header-lockup.png for the website navigation/header.

Use btf-logo-main-stacked-lockup.png for large brand moments such as the homepage hero or about page.

Use btf-logo-icon-mark.png for favicon, loading animation, social profile image, app-style icon, and small UI marks.

Use btf-logo-concept-board-winner.png only as visual reference for Cursor. Do not use the full concept board as a live website asset.

Keep the electric blue/cyan brand color direction. Do not replace it with a generic finance color palette.

The logo should feel operator-led, premium, modern, industrial, and trustworthy.
